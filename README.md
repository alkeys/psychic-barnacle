# Bienvenida al Proyecto Innova

Aquí encontrarás los comandos para correr el proyecto.

## Comandos

```bash
# Instalar dependencias
npm install

# Ejecutar el proyecto en desarrollo
pnpm run dev          

# Construir para producción
pnpm run build

# Ejecutar pruebas
npm test
```

## Requisitos

- Node.js >= 14
- npm >= 6
